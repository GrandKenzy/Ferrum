**Speed. Power. Security. Control.**

## Project Goals

The objective of Ferrum is to develop a programming language that is:

1. **Competent**: able to perform low-level tasks with maximum efficiency.  
2. **Attractive**: clear, modern, and human-readable syntax inspired by Python.  
3. **Powerful**: capable of compiling directly to machine code without heavy runtime dependencies.  
4. **Secure and Controllable**: protecting intellectual property while promoting internal learning and collaboration.
